# Sensory — outreach site

Public site for **Sensory**: sensor-agnostic predictive maintenance for
mid-size industry. Learns each machine's normal from its own history —
no failure labels — and reports deviations in plain language.

**Live:** https://freakysauce.github.io/sensory-site/

- `index.html` — the one-page site (fully self-contained, zero external requests)
- `demo/` — playable fleet dashboard: 30 NASA C-MAPSS turbofan engines the
  model never saw in training, ageing in calendar time (see `demo/README.md`)
